These tools run from the terminal and can autonomously write code, run commands, and manage files — all using CBorg as the AI backend.

## Aider

[Aider](https://aider.chat/) is a terminal-based AI pair programmer that works with your local git repository to make code changes.

→ [Using Aider with CBorg](/tools_aider)

## Claude Code

[Claude Code](https://docs.anthropic.com/en/docs/claude-code) is Anthropic's agentic CLI tool for software engineering tasks, configured to use CBorg instead of Anthropic's cloud.

→ [Claude Code with CBorg](/tools_claudecode)

## Gemini CLI

[Gemini CLI](https://github.com/google-gemini/gemini-cli) is Google's open-source AI agent for the terminal, pointed at CBorg's OpenAI-compatible endpoint.

→ [Gemini CLI with CBorg](/tools_geminicli)

## Codex CLI

[Codex CLI](https://github.com/openai/codex) is OpenAI's lightweight coding agent for the terminal, configured to use CBorg.

→ [Using CBorg with OpenAI Codex CLI](/tools_codex)

## Opencode

[Opencode](https://opencode.ai/) is an open-source terminal AI coding agent with a TUI interface.

→ [Opencode with CBorg](/tools_opencode)
