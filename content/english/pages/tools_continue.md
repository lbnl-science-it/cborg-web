#### VS Code Continue Plugin

Use the popular [VS Code Continue Plugin](https://www.continue.dev/) with the CBorg API Server.

Three installation methods are provided:

1. **Manual**: It will prompt you to enter an API key, which is then hard-coded into the config file
2. **Standard (Recommended)**: API key must be set in the environment. Config file is symlinked to this repo for updates.
3. **Proxy (Advanced)**: For CBorg Client-side Proxy users - improved performance on VPN or LBL-net
