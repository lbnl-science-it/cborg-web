## WARNING

Due to low usage as of 2026-02-01 we are no longer maintaining documentation and setup instructions for the Continue extention. The following is provided for historical context in case it is useful.


#### VS Code Continue Plugin

Using [VS Code Continue Plugin](https://www.continue.dev/) with the CBorg API Server.

Three installation methods are provided:

1. **Manual**: It will prompt you to enter an API key, which is then hard-coded into the config file
2. **Standard (Recommended)**: API key must be set in the environment. Config file is symlinked to this repo for updates.
3. **Proxy (Advanced)**: For CBorg Client-side Proxy users - improved performance on VPN or LBL-net (NO LONGER SUPPORTED)
